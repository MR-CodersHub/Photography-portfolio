Project Name
Lumina – Photography Portfolio Web Application

Short Description
Lumina is a responsive photography portfolio web application designed to showcase visual work through a clean, elegant, and media-focused interface. The application emphasizes visual storytelling, structured galleries, and smooth user interactions, supported by backend functionality for dynamic content handling.

Tech Stack

Tailwind CSS – Provides a modern, responsive layout optimized for visual presentation
Vanilla JavaScript – Enables client-side interactivity and dynamic gallery behavior
PHP – Handles backend logic and server-side operations
